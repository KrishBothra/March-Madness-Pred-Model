## Data Sources

**Primary Dataset:**
- [March Madness Data – Kaggle (nishaanamin)](https://www.kaggle.com/datasets/nishaanamin/march-madness-data/data?select=RPPF+Ratings.csv)

This Kaggle dataset aggregates data from the following sources:
- [Evan Miya](https://evanmiya.com/)
- [KenPom](https://kenpom.com/)
- [Barttorvik](https://www.barttorvik.com/#)
- [Z Rating](https://x.com/Z_rating)
- [ESPN](https://www.espn.com/)
- [Team Rankings](https://www.teamrankings.com/)
